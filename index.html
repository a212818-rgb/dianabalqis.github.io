Short Introduction

Kind Bites is a food donation and shelter finder app that allows users to contribute to food donation, find shelters, and join donation campaigns. The app provides an easier way for users to support communities in need by connecting them with food-related initiatives and shelter information. It also supports SDG 2: Zero Hunger by making food donation activities more accessible, organized, and community-focused through mobile technology.

Project 2 — Kind Bites with API, Sensors and Cloud

Problem Statement

Donors need a faster and easier way to find nearby shelters and submit food donation details. Shelters also need updated information about available donations. Kind Bites aims to solve this by using API integration, sensors and cloud storage.

Project 2 Screens

API Integration

Kind Bites uses API integration to provide real-time and location-based services. The app can use a Maps API or Location API to display nearby shelters, calculate distance and provide directions. A Donation API can be used to submit food donation details such as food type, quantity, expiry date and pickup location.

Sensors

Kind Bites uses the phone GPS or location sensor to detect the user’s current location when searching for nearby shelters. The camera sensor can also be used to capture food images as proof or reference before submitting a donation.

Cloud

Kind Bites can use Firebase as a cloud service to store user details, shelter information, donation records and uploaded images. Cloud storage allows the information to be updated and accessed more efficiently.
